Customers registration

This is a personal project for a barbershop, the client requested a web app to be able to carry out a basic record of his clients in order to create a file highlighting the possibility of taking notes of the work done and the date of the last visit. He also requested a search bar to filter the records and the possibility of seeing a detail of each client. It has a login with data base wich also keeps the records added by the user. This project is still in development, since it intends to be the main page of the barbershop later. The project is carried out with React, JS, Html, CSS, Firebase, Bootstrap.
